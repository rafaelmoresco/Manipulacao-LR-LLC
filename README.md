# Manipulacao-LR-LLC

Ideia de input:
    AF

    _|a|b
    ->A|B|C
    B|B|C
    *C|A|C

    Se o estado é composto, ex: {q0,q1}, deve estar em ordem crescente e escrito sem as chaves, ex.: q0,q1. Escrever em letras minúsculas, as letras maiúsculas foram reservadas para o processamento.

    Gramatica

    ->S,aA,bB
    A,aA,b
    B,aA,bA
